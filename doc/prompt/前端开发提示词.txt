现在开始开发数据中心运行监控大屏前端。
技术：
Vue3
Vite
ECharts
Axios

要求：
创建frontend项目。
实现：
1. 全屏大屏布局
2. 科技感UI设计
3. 深色背景
4. 半透明玻璃面板
5. 网格背景
6. 动态数据效果

页面设计要求：
请实现一个 1920x1080 风格的数据大屏页面，能够自适应浏览器窗口。

页面主题：
科技感、深色背景、紫色视觉风格，但不要只使用单一颜色，可配合金色、绿色进行使用。整体要专业、清晰、适合演示。

进阶设计：
UI 设计需要包含：
半透明玻璃质感数据面板（Glass Panel），搭配模糊、渐变和微弱发光效果；
科幻 HUD 边框设计，包括四角切割边框（Corner Bracket Border）、动态扫描线（Scanning Line）、坐标标记（Coordinate Marker）、数据标签（Data Label）等未来科技元素；
使用霓虹发光边框效果，通过 CSS box-shadow、linear-gradient、conic-gradient、pseudo-element 和 animation 实现动态光效、流动边框和能量环绕效果；
增加动态数据流、粒子流线、脉冲节点网络等背景动画效果；
增加细节动画，如呼吸灯效果、脉冲节点、数据刷新动画、扫描动画、粒子漂浮效果，使整个大屏具有动态生命感。

页面结构：
顶部：
EneDataCenterMonitor
第一行：
KPI数字卡片
显示：
服务器数量
指标数量
CPU平均利用率
内存使用率
中间区域：
CPU实时趋势折线图
内存趋势图
左侧：
服务器性能排行
右侧：
机房统计
底部：
网络流量趋势
磁盘IO趋势
使用：
ECharts
要求：
生成完整Vue代码。

以上页面结构只是推荐，请你作为一个专业的开发工程师，如果还有需要补充显示在大屏上的，请灵活设计页面，，最终效果清晰美观即可。
