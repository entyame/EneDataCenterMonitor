## 三、mod_detail.dat — 监控指标字典

**数据规模：** 55 个指标（20个性能类 + 35个磁盘类）

### 3.1 性能指标 (pref) — 20 个

| mod_id | 含义 | 单位 | tag |
|--------|------|------|-----|
| `cpu_user` | 用户态CPU使用率 | % | cpu_percent |
| `cpu_sys` | 系统态CPU使用率 | % | cpu_percent |
| `cpu_wait` | IO等待CPU使用率 | % | cpu_percent |
| `cpu_idle` | CPU空闲率 | % | cpu_percent |
| `cpu_usage` | CPU综合使用率 | % | cpu_percent |
| `mem_used` | 已使用内存 | MB | mem_metric |
| `mem_free` | 空闲内存 | MB | mem_metric |
| `mem_buff` | 缓冲区内存 | MB | mem_metric |
| `mem_cache` | 缓存内存 | MB | mem_metric |
| `mem_swap` | 交换区使用 | MB | mem_metric |
| `net_in` | 网络入站带宽 | MB/s | net_speed_mb |
| `net_out` | 网络出站带宽 | MB/s | net_speed_mb |
| `net_pktin` | 每秒入站数据包 | pkt/s | net_packets |
| `net_pktout` | 每秒出站数据包 | pkt/s | net_packets |
| `load1` | 1分钟平均负载 | — | load_average |
| `load5` | 5分钟平均负载 | — | load_average |
| `load15` | 15分钟平均负载 | — | load_average |
| `proc_run` | 运行中进程数 | 个 | proc_count |
| `proc_block` | 阻塞进程数 | 个 | proc_count |
| `proc_total` | 总进程数 | 个 | proc_count |

### 3.2 磁盘指标 (disk) — 35 个（5块磁盘 × 7个指标）

磁盘名：`sda`、`sdb`、`sdc`、`sdd`、`sde`

| 指标后缀 | 含义 | 单位 | tag |
|----------|------|------|-----|
| `_rqm` | 每秒合并读请求数 | req/s | disk_rqm_per_sec |
| `_read` | 每秒读取扇区数 | sectors/s | disk_rw_sectors |
| `_write` | 每秒写入扇区数 | sectors/s | disk_rw_sectors |
| `_avgrq` | 平均请求扇区大小 | sectors | disk_other_metric |
| `_await` | 平均I/O等待时间 | ms | disk_latency_ms |
| `_svctm` | 平均服务时间 | ms | disk_latency_ms |
| `_util` | 磁盘使用率 | % | disk_util_percent |

## 四、pref_tsar.dat — 性能监控事实数据

| # | 字段名 | 数据类型 | 含义 | 示例值 |
|---|--------|----------|------|--------|
| 1 | `ts` | BIGINT | 毫秒级Unix时间戳 | `1782835200000` |
| 2 | `hostid` | VARCHAR(20) | 主机ID | `host001` |
| 3 | `type` | VARCHAR(10) | 固定值 `pref` | `pref` |
| 4 | `mod` | VARCHAR(30) | 指标编码 | `cpu_usage` |
| 5 | `value` | DOUBLE | 指标值 | `43.87` |
| 6 | `tag` | VARCHAR(30) | 指标分组标签 | `cpu_percent` |

- **时间范围：** 2026-06-30 ~ 2026-07-06（7天）
- **采样频率：** 每小时一条，共 168 个时间点
- **覆盖主机：** 20 台全量覆盖
- **数据规模：** 20主机 × 20指标 × 168小时 = 67,200 行

| # | 字段名 | 数据类型 | 含义 | 示例值 |
|---|--------|----------|------|--------|
| 1 | `ts` | BIGINT | 毫秒级Unix时间戳 | `1782835200000` |
| 2 | `hostid` | VARCHAR(20) | 主机ID | `host003` |
| 3 | `type` | VARCHAR(10) | 固定值 `disk` | `disk` |
| 4 | `mod` | VARCHAR(30) | 指标编码 | `sda_write` |
| 5 | `value` | DOUBLE | 指标值 | `280043` |

- **采样方式：** 稀疏采样（每个时间点仅1条记录，随机主机/指标）
- **数据规模：** 12,000 行

## 六、数据关联关系

```
host_detail.hostid  ←→  pref_tsar.hostid     (1 : N)
host_detail.hostid  ←→  disk_tsar.hostid     (1 : N)
mod_detail.mod      ←→  pref_tsar.mod        (1 : N)
mod_detail.mod      ←→  disk_tsar.mod        (1 : N)
```

- **dim_host**：主机维度表（20行）
- **dim_mod**：指标字典维度表（55行）
- **fact_pref_tsar**：性能事实表（67,200行，**星型模型**核心事实表）
- **fact_disk_tsar**：磁盘事实表（12,000行，**星型模型**辅助事实表）
