# 大屏指标设计文档 — 数据中心运行监控大屏

> 对应 Vue + ECharts 可视化大屏的数据需求规格

---

## 一、大屏布局（建议）

```
┌─────────────────────────────────────────────────────┐
│              数据中心运行监控大屏                      │
├──────────┬──────────┬──────────┬──────────┬──────────┤
│ 服务器总数 │ 在线服务器 │ CPU使用率 │ 内存使用率 │ 网络流量  │   ← 顶部 KPI 数字卡片
├──────────┴──────────┴──────────┴──────────┴──────────┤
│                                                       │
│  ┌──────────────┐  ┌──────────────┐  ┌─────────────┐ │
│  │ CPU 24h趋势  │  │ 内存 24h趋势 │  │ 网络流量趋势 │ │   ← 中间：趋势折线图
│  └──────────────┘  └──────────────┘  └─────────────┘ │
│  ┌──────────────┐  ┌──────────────┐  ┌─────────────┐ │
│  │ 负载趋势     │  │ 磁盘IO趋势   │  │ 磁盘利用率   │ │
│  └──────────────┘  └──────────────┘  └─────────────┘ │
│                                                       │
│  ┌─────────────────────┐  ┌─────────────────────────┐│
│  │  机房服务器分布饼图   │  │  CPU TOP10 横向柱状图    ││   ← 底部：分布 & 排名
│  └─────────────────────┘  └─────────────────────────┘│
│  ┌─────────────────────┐  ┌─────────────────────────┐│
│  │  磁盘TOP10 柱状图    │  │  实时告警滚动列表        ││
│  └─────────────────────┘  └─────────────────────────┘│
│                                                       │
└─────────────────────────────────────────────────────┘
```

## 二、指标清单与数据源映射

### A. 首页核心指标 (KPI Cards)

| 序号 | 指标名称 | 数据表 | SQL字段 | 图表组件 | 刷新频率 |
|------|---------|--------|---------|----------|----------|
| A1 | 服务器总数 | `dashboard_kpi_summary` | `kpi_name='host_online'` | StatCard 数字卡片 | 1小时 |
| A2 | CPU综合使用率 | `dashboard_kpi_summary` | `kpi_name='cpu_usage'` | Gauge 仪表盘 | 1小时 |
| A3 | 内存使用量 | `dashboard_kpi_summary` | `kpi_name='mem_used'` | Progress 进度条 | 1小时 |
| A4 | 网络入站带宽 | `dashboard_kpi_summary` | `kpi_name='net_in'` | StatCard | 1小时 |
| A5 | 磁盘使用率 | `dashboard_kpi_summary` | `kpi_name='disk_util'` | Gauge 仪表盘 | 1小时 |

### B. 趋势分析 (Trend Lines)

| 序号 | 图表名称 | 数据表 | 维度字段 | 指标字段 | 图表类型 |
|------|---------|--------|---------|---------|----------|
| B1 | CPU 24h趋势 | `dashboard_cpu_hour` | `ts_date, ts_hour` | `cpu_user/sys/wait/usage/idle_avg` | 多线折线图 |
| B2 | 内存 24h趋势 | `dashboard_mem_hour` | `ts_date, ts_hour` | `mem_used/free/buff/cache/swap_avg` | 堆叠面积图 |
| B3 | 网络流量趋势 | `dashboard_net_hour` | `ts_date, ts_hour` | `net_in/out_avg` | 双轴折线图 |
| B4 | 负载趋势 | `dashboard_load_hour` | `ts_date, ts_hour` | `load1/5/15_avg` | 多线折线图 |
| B5 | 磁盘IO趋势 | `dashboard_disk_hour` | `ts_date, ts_hour` | `disk_read/write/avg, util_avg` | 折线图 |

### C. 排名分析 (TOP Rankings)

| 序号 | 图表名称 | 数据表 | 筛选条件 | 图表类型 |
|------|---------|--------|---------|----------|
| C1 | CPU TOP10 | `dashboard_host_rank` | `rank_type='cpu' AND rank_position <= 10` | 横向柱状图 |
| C2 | 内存 TOP10 | `dashboard_host_rank` | `rank_type='memory' AND rank_position <= 10` | 横向柱状图 |
| C3 | 磁盘 TOP10 | `dashboard_host_rank` | `rank_type='disk' AND rank_position <= 10` | 横向柱状图 |

### D. 分布分析 (Distribution)

| 序号 | 图表名称 | 数据表 | 图表类型 |
|------|---------|--------|----------|
| D1 | 机房服务器分布 | `dashboard_room_summary` | 饼图 (Pie) |
| D2 | 各磁盘利用率对比 | `fact_disk_tsar` (直接查) | 柱状图 (Bar) |

### E. 异常告警 (Alerts)

| 序号 | 功能 | 数据表 | 筛选条件 | 图表类型 |
|------|------|--------|---------|----------|
| E1 | CPU高负载告警 | `dashboard_alert_detail` | `alert_type='cpu'` | 滚动列表（红色高亮） |
| E2 | 磁盘高IO告警 | `dashboard_alert_detail` | `alert_type='disk'` | 滚动列表（橙色高亮） |

## 三、前端 API 建议

Vue 前端通过 RESTful API 查询结果表，示例：

```
GET /api/dashboard/kpi          → SELECT * FROM dashboard_kpi_summary
GET /api/dashboard/cpu-trend    → SELECT * FROM dashboard_cpu_hour ORDER BY ts_date, ts_hour
GET /api/dashboard/host-rank?type=cpu&top=10 → SELECT * FROM dashboard_host_rank WHERE rank_type='cpu' AND rank_position <= 10
GET /api/dashboard/alerts       → SELECT * FROM dashboard_alert_detail ORDER BY alert_level, current_value DESC
```

## 四、告警阈值配置

| 告警类型 | 阈值 | 告警级别 | 说明 |
|----------|------|----------|------|
| CPU使用率 | > 80% | CRITICAL | 需立即处理 |
| CPU使用率 | 60% ~ 80% | WARNING | 需关注 |
| 磁盘使用率 | > 70% | WARNING | 需关注 |
| 内存使用量 | > 90,000 MB | WARNING | 接近饱和 |
